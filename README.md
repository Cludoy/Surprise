# 🌹 Surprise A JoJo's Bizarre Birthday Adventure

A spectacular, interactive **Happy Birthday** website themed around **JoJo's Bizarre Adventure**, built as a surprise gift for Gil. Hosted on GitHub Pages.

## ✨ Features

- **ZA WARUDO Time Stop** — Click the badge to freeze time, complete with a running clock and purple particle effects
- **Stand Rush Cries** — ORA ORA ORA / MUDA MUDA / DORA DORA — click the stand cards to trigger dramatic overlay text
- **Confetti Stand Rush** — The finale button detonates a golden confetti explosion with MUDA MUDA typing animation
- **Cursor Sparkles** — Move your mouse for floating stars and roses ✨
- **Scroll Reveal** — Elements animate in as you scroll
- **Konami Code Easter Egg** — `↑↑↓↓←→←→BA` triggers REQUIEM! 🌟
- **Manga Aesthetics** — Halftone dot backgrounds, speed lines, floating roses, golden shimmer typography
- **Fully Responsive** — Works on desktop and mobile

## 🚀 Live Site

👉 **[https://cludoy.github.io/Surprise](https://cludoy.github.io/Surprise)**

## 🛠 Tech Stack

- Pure HTML5, CSS3, vanilla JavaScript
- Google Fonts: Bangers, Bebas Neue, Black Han Sans, Inter
- No dependencies — works straight from the browser

## 📦 Deployment (GitHub Pages)

This site is automatically served via GitHub Pages from the `main` branch root.

To update: push changes to `main` and GitHub Pages will redeploy automatically.
